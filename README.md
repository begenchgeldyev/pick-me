# PickMe

Мобильное приложение на React Native.

## Быстрый старт

Требования:
- Node.js `>= 22.11.0`
- Настроенная среда React Native (Android Studio/Xcode)

Установка и запуск Metro:

```bash
npm install
npm start
```

Запуск приложения (в отдельном терминале):

```bash
npm run android
# или
npm run ios
```

## Полезные команды

```bash
npm run lint
npm test
```
