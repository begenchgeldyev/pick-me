# Приложение PickMe

Мобильное приложение для поиска попутчиков (водителя/пассажира) для длительных поездках на авто. 
Язык разработки - React Native.

## Начало работы

Требования:
- Node.js `>= 22.11.0`
- Настроенная среда React Native (Android Studio/Xcode)

Установка и запуск Metro:

```bash
npm install # только при самом первом запуске
npm start
```

Запуск приложения (в другом терминале):

```bash
# для android приложения
npm run android
# для ios приложения
npm run ios
```

## Другие полезные команды

```bash
npm run lint #показать логи
npm test # команда используется для запуска тестов проекта при локальном тестировании или при развёртывании на удалённом сервере 
```
## Сценарий для проверки 
Регистрация пользователя
1. Запустите приложение и откройте эмулятор Android Studio/Xcode
2. На главном экране приложения в эмуляторе нажимте "Регистрация"
3. Вам должно открыться приветсвенное окно, нажмите в нем "Поехали"
4. Далее окно с возможностями приложения
5. Нажмите в нем "Дальше"
6. Повторите шаг 4 со всеми следующими приветственными окнами
7. Вам откроется окно регистрации
8. Заполните все обязательные поля в открывшемся окне регистрации
9. Нажмите "Зарегистрироваться"
10. Откроется страница с полем для ввода кода подтверждения
11. Введите любой код (потом будет не любой)
12. Вам должно открыться окно с уведомлением об успешной регситрации и кнопкой перехода в приложение
